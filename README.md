# n-wrapper

Make the [n](https://github.com/tj/n) easier to use on fish

## Install

With [fisherman]

```
fisher robsonpeixoto/n-wrapper
```

## Notes

`n-wrapper` was built for the latest fish.

## Usage

### List all installed versions

```fish
n-ls
```

### Activating a version

```
n-workon 6.3.0
```

### Deactivating a version

```
n-deactivate
```

### To install and remove a node version

```
n 6.3.0    # to install
n rm 6.2.0 # to remove
```
